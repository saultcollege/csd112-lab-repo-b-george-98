Glossary

Elements:
Contain opening tags, closing tags, with content in between. What form the basic skeleton of a website.
Attribute:
Gives the element more information.
Whitespace:
String of text only made up of tabs, spaces, or line breaks.
Character Set:
Tells a web browser the character set to use.
Metadata:
Data about data.
